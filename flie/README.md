# 开发常用文件后缀名

```
.babelrc                //babel编译文件
.eslintignore           //eslint校验过滤文件
.eslintrc               //eslint属性设置文件
.flowconfig             //flowjs静态语法检测
.gitignore              //git过滤文件
.gitmodules             //git子项目文件
.yml                    //git-ci文件
webpack.config.js       //webpack打包文件
package.json            //npm项目依赖文件
README.md               //项目说明文件
```
